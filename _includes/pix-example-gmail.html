Creating a Gmail account - Desktop

The following is an example of the interaction flow to create a new Gmail account.

    • User needs a new gmail account
    • He visits gmail.com on his desktop
    • Clicks on 'Create an account'
    • System loads a new page
    • He needs to fill the form
    • Basic info such as name and user name
    • Jump to next input using 'TAB'
    • Guidance to choose an appropiate password
    • Messages appear when clicking on every input
    • This is nice and easy
    • Helpful tooltips support the process
    • Privacy policy and Service condition
    • Information required marked as red
    • Clicks on "Next step"
    • System verifies the information and saves it
    • The user is invited to fill his Google Plus profile
    • Clicks on 'No, thanks' button
    • The account has been created. Yay!
    • Welcome page with basic account info
